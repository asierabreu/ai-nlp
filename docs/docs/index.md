## ⚡ Quick Start

### Run the Docs Locally

```bash
cd docs
pip install -r requirements.txt
mkdocs serve
# Visit http://localhost:8000
```

### Docker

```bash
docker-compose up --build docs
# Visit http://localhost:8000
```
